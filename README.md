# Детсад Управление - Фронтенд

Веб-приложение для управления детским садом, предоставляющее удобный интерфейс для администраторов, воспитателей и других сотрудников.

## Архитектура

Фронтенд-приложение построено на React с использованием TypeScript. Архитектура приложения следует современным практикам React-разработки с компонентным подходом и разделением ответственности.

### Структура проекта:
- `src/app/` - основные компоненты приложения (макеты, навигация)
- `src/modules/` - модули приложения, каждый со своей логикой
- `src/entities/` - сущности приложения (дети, группы, сотрудники и т.д.)
- `src/shared/` - переиспользуемые компоненты, хуки, утилиты и сервисы

### Технологии:
- Vite 7
- React 18
- TypeScript
- Material UI 5 (MUI) + Emotion
- React Router 6 (навигация)
- Axios (API запросы)
- React Context API (управление состоянием)
- Recharts (графики и интерактивная статистика)
- Библиотеки: xlsx, docx, Framer Motion (анимации), notistack, Sentry
- Yandex Maps API (геолокация сотрудников)
- Service Workers (PWA функционал)
- Интеграция с AI Engine (интерфейс чата и анализа)

## Управление состоянием (Contexts)

В приложении используется **React Context API** для глобального состояния:
- **AuthContext**: Авторизация, данные пользователя и права доступа.
- **GroupsContext**: Список групп и управление текущей выбранной группой.
- **DateContext**: Текущая рабочая дата для фильтров.
- **LayoutContext**: Состояние бокового меню и макета.
- **TimeTrackingContext**: Статус чекина/чекаута сотрудника.

## Развёртывание

Проект развёртывается на **Vercel**. Конфигурация находится в \`vercel.json\`.
Для работы SPA на Vercel настроены редиректы всех путей на \`index.html\`.

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: \`npm install\`
3. Настройте \`.env\` (см. выше)
4. Запустите dev-сервер: \`npm run dev\`
5. Сборка для продакшена: \`npm run build\`

Приложение будет доступно по адресу http://localhost:5173 (по умолчанию для Vite)
